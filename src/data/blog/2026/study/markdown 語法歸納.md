## 前言

只篇文章用來歸納寫博客會用著個Markdown語法，方便以後查閱。

---

## 一、標題

```markdown
# 一級標題
## 二級標題
### 三級標題
#### 四級標題
##### 五級標題
###### 六級標題
```

一般文章用二級、三級標題就夠，一級標題留畀文章題目用。

---

## 二、文字樣式

```markdown
**粗體文字**
*斜體文字*
***粗斜體***
~~刪除線~~
`行內代碼`
```

效果：

**粗體文字** ｜ *斜體文字* ｜ ***粗斜體*** ｜ ~~刪除線~~ ｜ `行內代碼`

---

## 三、段落同換行

```markdown
只係第一段。

只係第二段，空一行就係新段落。

若是欲強制換行，  
行尾加兩個空格。
```

---

## 四、列表

### 無序列表

```markdown
- 第一項
- 第二項
  - 子項目（前面加兩格空白）
  - 子項目二
- 第三項
```

效果：

- 第一項
- 第二項
  - 子項目
  - 子項目二
- 第三項

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

效果：

1. 第一步
2. 第二步
3. 第三步

### 任務列表

```markdown
- [x] 已完成個任務
- [ ] 未完成個任務
- [ ] 另一個任務
```

效果：

- [x] 已完成個任務
- [ ] 未完成個任務
- [ ] 另一個任務

---

## 五、連結同圖片

### 連結

```markdown
[連結文字](https://example.com)

[帶標題個連結](https://example.com "鼠標懸停顯示")

直接顯示網址：<https://example.com>
```

### 圖片

```markdown
![圖片描述](圖片路徑)

![本地圖片](../../assets/images/example.jpg)

![網絡圖片](https://example.com/image.jpg)
```

## 六、引用

```markdown
> 只係一段引用文字。

> 多行引用
> 繼續引用

> 嵌套引用
>> 內層引用
```

效果：

> 只係一段引用文字。

> 多行引用
> 繼續引用

> 嵌套引用
>> 內層引用

---

## 七、代碼

### 行內代碼

```markdown
用反引號包住 `代碼` 就好。
```

### 代碼塊

````markdown
```javascript
const greeting = "汝好！";
console.log(greeting);
```

```python
def greet():
    print("汝好！")
```

```css
body {
  background-color: #000;
}
```
````

效果：

```javascript
const greeting = "汝好！";
console.log(greeting);
```

```python
def greet():
    print("汝好！")
```

支持語法高亮個語言包括：`javascript`、`typescript`、`python`、`css`、`html`、`bash`、`json`、`markdown` 等等。

## 八、表格

```markdown
| 標題一 | 標題二 | 標題三 |
|--------|:------:|-------:|
| 靠左   | 居中   | 靠右   |
| 內容   | 內容   | 內容   |
```

效果：

| 標題一 | 標題二 | 標題三 |
|--------|:------:|-------:|
| 靠左   | 居中   | 靠右   |
| 內容   | 內容   | 內容   |

`|--------|` 靠左，`|:------:|` 居中，`|-------:|` 靠右。

## 十、跳脫字符

若是欲顯示 Markdown 個特殊符號，用反斜杠 `\` 跳脫：

```markdown
\*唔係斜體\*
\# 唔係標題
\[唔係連結\]
```

效果：\*唔係斜體\* ｜ \# 唔係標題

---

## 十一、Frontmatter 字段

每篇文章頂部個 `---` 內容：

```yaml
---
title: 文章標題           # 必填
author: 作者名字          # 必填
pubDatetime: 2026-04-05T10:00:00Z  # 發佈時間，必填
modDatetime: 2026-04-05T12:00:00Z  # 修改時間，選填
description: 文章描述     # 必填，顯示在列表頁
tags:                     # 選填
  - 標籤一
  - 標籤二
featured: false           # 是否精選，顯示在首頁頂部
draft: false              # true 為草稿，唔會發佈
ogImage: ""               # 分享圖片，選填
canonicalURL: ""          # 規範網址，選填
---
```
